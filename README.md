# Freaky Fashion

A responsive e-commerce web application developed as part of the JavaScript 2 course in my Frontend Developer education.

## About

Freaky Fashion is a full-featured fashion e-commerce application built with React. The project includes product browsing, search functionality, dynamic product pages, shopping cart management, checkout flow, and an administration interface for managing products.

The application was developed according to advanced project requirements and received the highest grade (VG).

## Features

* Product listing generated from database content
* Dynamic product detail pages
* Product search and search results page
* Shopping cart with quantity management
* Checkout page
* Responsive design for mobile, tablet, and desktop
* Product administration panel
* Create new products
* Delete existing products
* Form validation

## Built With

* React
* JavaScript
* HTML5
* CSS3
* React Router
* REST API
